由于 Mod Jam 3 (v1.2.0) 暂时不支持翻译。必须替换文本的xml文件，请找到Mod Jam 3模组的文件夹，并把
modjam3/planets 文件夹与 xen.ModJam3/planets 合并。注意版本问题。
